## Tecnologías

- HTML5 semántico
- CSS3 con variables personalizadas y Flexbox

## Estructura del proyecto
```
edulearn/
├── EduLearn.html # Página principal
├── css/
│ ├── styles.css # Estilos principales
│ └── css2.css # Estilos secundarios
└── assets/ # Imágenes y recursos
```
## Uso

Abre `EduLearn.html` directamente en el navegador o usa **Live Server** en VS Code.

```bash
git clone https://github.com/maryhug/edulearn.git
```
